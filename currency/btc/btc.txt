3.bit docket
    3.1 run in docker
        docker run -it -p 19001:19001 -p 19011:19011 freewil/bitcoin-testnet-box
        make start
    3.2 look message
        make getinfo
    3.3 产生一个区块
        make generate
    3.4 产生100个区块
        make generate BLOCKS=100
    3.5 查看节点状态
        bitcoin-cli -datadir=1  getinfo
    3.6 查看钱包地址
        bitcoin-cli -datadir=1  getwalletinfo
        bitcoin-cli -datadir=1 getnewaddress
